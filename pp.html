PowerPoint Gesture Controller

https://github.com/zuhaakashif/PowerPoint-Controller

A Python-based system that lets you control PowerPoint slides using hand gestures detected via webcam — no clicker or keyboard needed.

The PowerPoint Gesture Controller leverages computer vision to make presentations hands-free. Using MediaPipe for real-time hand landmark detection and OpenCV for video processing, the system tracks fist gestures to move slides back and forth. Gestures are translated into simulated keypresses via PyAutoGUI, allowing users to control PowerPoint smoothly with physical cues — making demos more interactive and fluid.

Traditional presentation tools—like clickers or keyboards—can tether presenters, limiting movement or making transitions feel clunky. The challenge was to build a gesture-based interface that’s responsive, intuitive, and low-cost, enabling presenters to navigate slides naturally with hand motions.

The project creates a more engaging presentation experience by enabling hands-free navigation, allowing presenters to be more dynamic and expressive. It’s especially useful in educational settings, live demos, or remote presentations where mobility and flow matter.

Key Features
