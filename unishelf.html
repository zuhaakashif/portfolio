UniShelf – Student Course Material Finder App

Github Link

UniShelf is an Android app built with Kotlin that helps students easily access semester-specific study resources. It centralizes books, notes, and past papers in one organized place with Firebase login support.

UniShelf is a Kotlin-based Android application designed to simplify academic life for university students. Built with a clean Material Design interface, it allows users to explore and access semester-specific course materials—such as books, past papers, course outlines, and notes—seamlessly in one place. It also supports user authentication via Firebase and offers a handy dark mode for easier reading.

Students often struggle with managing and locating scattered course materials, especially when resources are spread across multiple platforms or folders. The goal was to create a centralized, intuitive app where students can quickly find everything relevant to their current semester.

UniShelf streamlines the academic experience by putting all course essentials at students’ fingertips. With seamless login, semester filtering, and dark mode viewing, it helps learners stay organized, focused, and less overwhelmed—especially during late-night study sessions.

Key Features

  • Firebase Authentication
  • Semester-based Resource Access
  • Centralized Repository
  • Material Design UI
  • Dark Mode
